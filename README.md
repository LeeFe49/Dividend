# Dividend
*** 배당금 프로젝트 ***

### 프레임워크 & 라이브러리
- java 11
- Gradle 7.2
- Spring Boot 2.5.6
- h2 DB
- Jsoup

### API
- 회사 정보 저장
- 저장된 회사 정보 삭제
- 회사의 배당금 정보 조회
- 회원가입 및 로그인/로그아웃
- 사용자 인증
- 검색 자동완성
